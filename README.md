BookSwap — Платформа для обмена книгами!

BookSwap — это веб-приложение на Ruby on Rails, котрое позволяет пользователям выставлять свои книги на обмен и находить новые книги у других участников сообщества.
Основные функции
    Система профилей: Регистрация и авторизация пользователей с использованием зашифрованных паролей
    Каталог книг: Возможность добавлять, редактировать и удалять свои книги, а также просматривать книги других пользователей
    Механика обмена: * Отправка запроса на обмен с сопроводительным сообщением
    Личный кабинет со списками отправленных и полученных запросов
    Принятие или отклонение запросов владельцем книги
    При принятии запроса права собственности на книгу переходят к новому владельцу автоматически

стек
